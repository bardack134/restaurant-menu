

## Description:
このコードは、レストランのウェブサイトを管理するためのPHPスクリプトです。メニュー管理、予約管理、ナビゲーション、モーダル表示などを実装しています。データベースと連携し、画像や予約情報の追加・更新・削除が可能です。Bootstrapでデザインし、JavaScriptでインタラクションを制御します。

## Tools Used:
- html
- css
- bootstrap
- javascript
- php
- データベース
